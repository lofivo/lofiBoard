# lofiBoard

vibe的💩，自用

一个离线优先的可编辑网页白板。当前版本使用 Vite + Konva，支持无限画布、对象选择、画笔、双橡皮、文字、基础形状、本地白板文件打开/保存、背景模式切换和 PNG 导出。

## 开发

```bash
npm install
npm run dev
```

生产构建：

```bash
npm run build
npm run preview
```

构建产物在 `dist/`，可通过本地或静态服务器离线运行。

## 功能

- 无限画布：滚轮缩放，按住空格或中键拖拽平移。
- 工具：底部图标工具栏提供选择、画笔、片段橡皮、对象橡皮、文字、矩形、椭圆、线段、箭头。
- 图形：矩形、椭圆、线段、箭头合并在图形工具中，点击图形工具会弹出子工具选择。
- 编辑：单选/框选、多选、移动、缩放、旋转、删除、置顶、置底，上下文属性面板会随工具或选中对象显示。
- 样式：支持线条颜色、填充颜色、透明填充、粗细和文字字号。
- 橡皮：片段橡皮支持动态擦除半径，快速移动时扩大，慢下来后回到基础大小。
- 文件：在 Chrome/Edge 中使用 `.lofibrd` JSON 文件原地打开、保存和另存为。
- 菜单：点击左上角白板名称或设置图标，可新建、打开、保存、另存为、导出 PNG、清空画布、切换背景、重置视图。
- 背景：支持点阵和纯白背景，设置会保存进 `.lofibrd` 文件。
- 导出：将当前视图按当前背景模式导出为 PNG。

## 快捷键

- `Ctrl+S`：保存
- `Ctrl+Shift+S`：另存为
- `Ctrl+O`：打开
- `Ctrl+Z`：撤销
- `Ctrl+Y`：重做
- `Delete` / `Backspace`：删除选中对象
- `Space`：按住平移画布
- `V`：选择
- `B`：画笔
- `E`：片段橡皮
- `O`：对象橡皮
- `T`：文字
- `R`：矩形
- `L`：线段
- `A`：箭头

## 目录结构

- `src/whiteboard-app.js`：应用状态和 Konva 事件编排。
- `src/app-shell.js`：白板工作台 HTML 结构。
- `src/ui-config.js`：工具、菜单和图标配置。
- `src/element-factory.js`：白板元素数据创建。
- `src/konva-elements.js`：白板元素到 Konva 节点的渲染适配。
- `src/file-service.js`：本地白板文件打开、保存和导出下载。
- `src/tool-behavior.js`：工具行为规则，例如动态橡皮和透明填充。
